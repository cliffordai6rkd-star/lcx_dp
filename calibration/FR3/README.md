## 1.2 二维码版本

这里提供三种方法来做
第一种，手动控制机械臂到位姿，该方法灵活性大，可以大幅度旋转机械臂来操作。
第二种，设定3D网络空间点，让机械臂自动移动到计算的空间点中。每次移动后摁下r,保存。如果二维码超出空间，按l跳过该点。
第三种，全自动。
二维码可能会由于环境的噪声产生随机的别的码出来，目前不建议使用全自动的方法。

### 1.2.1 使用方法

#### 1.2.1.1 第一种手动 eye_to_hand.py

运行程序，输入IP。将机械臂变为programming模式。移动机械臂（假如旋转，尽可能让每个关节都动一下），观察屏幕等坐标稳定，摁r保存。大约标记20-30张照片。按q推出。运行eye_to_hand_calibration.py 算出手眼标定的矩阵。

#### 1.2.1.2 第二种eye_to_hand_auto_r.py

运行程序，输入IP.输入你要设定的工作范围。X轴、Y轴、Zzhou 的范围。输入想要生成的点数个数，然后生成3D网格点。
然后按a开始标定。机械臂会走到第一个点。然后用户摁r,机械臂走下一个网络点。如果二维码超出空间，则按l跳过这个网络点。最后按q保存数据。 运行eye_to_hand_calibration.py 算出手眼标定的矩阵。

# TIPS
- 采集至少8-10个位姿，最好15-20个
- 确保三个旋转轴都有变化：
  - 末端绕X轴旋转约±30°
  - 末端绕Y轴旋转约±30°
  - 末端绕Z轴旋转约±45°
- 在不同距离处采集数据
- 在视野的不同位置捕捉标定板
